# 一键启动指南

## 🚀 快速启动方法

本项目提供了多种启动方式，无论项目被放在哪个目录下都能正确运行。

### macOS 用户

#### 方法1：双击启动（推荐）
1. 在Finder中找到项目目录
2. 双击 `启动KL-zhanghao.command` 文件
3. 如果是首次运行，会自动安装依赖（需要几分钟）
4. 安装完成后自动启动GUI

#### 方法2：终端启动
```bash
cd /path/to/KL-zhanghao
./start.sh
```

#### 创建桌面快捷方式
1. 右键点击 `启动KL-zhanghao.command`
2. 选择"制作替身"（Make Alias）
3. 将替身拖到桌面或Dock栏
4. 以后直接双击桌面图标即可启动

---

### Windows 用户

#### 方法1：双击启动（推荐）
1. 在文件资源管理器中打开项目目录
2. 双击 `启动KL-zhanghao.bat` 文件
3. 如果是首次运行，会自动安装依赖（需要几分钟）
4. 安装完成后自动启动GUI

#### 方法2：PowerShell启动
```powershell
cd C:\path\to\KL-zhanghao
.\start.ps1
```

#### 创建桌面快捷方式
1. 右键点击 `启动KL-zhanghao.bat`
2. 选择"发送到" → "桌面快捷方式"
3. 以后直接双击桌面图标即可启动

---

## 📋 启动文件说明

### macOS
- `启动KL-zhanghao.command` - 双击启动文件（带图标，可拖到Dock）
- `start.sh` - 主启动脚本

### Windows
- `启动KL-zhanghao.bat` - 双击启动文件（CMD批处理）
- `start.ps1` - 主启动脚本（PowerShell）

## ⚠️ 常见问题

### macOS

**Q: 双击.command文件没反应？**  
A: 需要设置执行权限：
```bash
chmod +x 启动KL-zhanghao.command
chmod +x start.sh
```

**Q: 提示"无法打开，因为无法验证开发者"？**  
A: 右键点击文件，选择"打开"，然后点击"打开"按钮。

---

### Windows

**Q: 提示"无法执行脚本，因为在此系统上禁用了脚本"？**  
A: 以管理员身份运行PowerShell，执行：
```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```

**Q: 双击.bat文件一闪而过？**  
A: 脚本已经添加了错误暂停功能，如果还是闪退，请右键"以管理员身份运行"。

---

## 🔧 手动安装依赖

如果自动安装失败，可以手动安装：

### macOS/Linux
```bash
bash scripts/install.sh
```

### Windows
```powershell
./scripts/install.ps1
```

## 📞 技术支持

遇到问题请检查：
1. Python版本是否为3.10或更高
2. 网络连接是否正常（安装依赖需要联网）
3. 是否有足够的磁盘空间（至少500MB）
4. 查看终端输出的错误信息
